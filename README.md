# 🌱 Jogo-Aventura-Agrinho

> Uma corrida interativa e acessível que utiliza a lógica de programação para conectar os jovens ao futuro da sustentabilidade no campo.

---

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido como parte do Concurso Agrinho 2026. O principal objetivo é transformar o aprendizado sobre o agronegócio em uma experiência divertida e competitiva, utilizando um jogo de corrida escalável que ensina lógica de desenvolvimento de forma prática, inclusiva e totalmente adaptável a qualquer dispositivo.

## 🛠️ Tecnologias Utilizadas
Para construir a estrutura, o design e as interações do site, foram aplicadas as seguintes tecnologias web:
* **HTML5:** Estruturação do conteúdo e acessibilidade.
* **CSS3:** Design responsivo (mobile-first), transições fluidas e efeitos visuais.
* **JavaScript / p5.js:** Criação do jogo de corrida utilizando estruturas dinâmicas de listas (arrays) e laços de repetição.

## ✨ Recursos de Interface e Acessibilidade
Preocupando-se com a experiência de todos os usuários, o projeto conta com:
* 📱 **Design Responsivo:** O site se adapta perfeitamente a celulares, tablets e computadores.
* 👁️ **Contraste de Cores:** Cores validadas para garantir uma leitura confortável e inclusiva (fundo escuro com elementos de alto contraste).
* 🎨 **Feedback Visual:** Interatividade mapeada pelo teclado com respostas visuais instantâneas na tela.

---

## 📁 Organização dos Arquivos
O projeto está estruturado de forma limpa e padronizada:
* `index.html` - Página principal do site.
* `style.css` - Estilização e regras de responsividade.
* `script.js` - Lógica de interatividade e o jogo de corrida em p5.js.
* `README.md` - Documentação do projeto (esta página).
